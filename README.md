# Build an API Proxy Server with node.js/express and a Filterable List

# API

https://proxy.homeran.co.il/videos

# Filterable List

https://videos.homeran.co.il


## Setup

```sh
cp server/.env.sample server/.env
```

Update .env with your API KEY.

## Start server

```sh
cd server
npm run dev
```

## Start client

```sh
cd client
npx lite-server
```
